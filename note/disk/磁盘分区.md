# 硬盘的分区、格式化和挂载

1. 对磁盘分区，创建可用partition（类比windows C，D盘符）
2. 对该partition格式化，创建filesystem （类比windows NTFS）
3. 创建挂载点

## 观察磁盘分区状态
**常用命令**
- `lsblk`  查看磁盘分区状态(list block devices)
-  分区工具 `fdisk` `gdisk` 
    **example input**
    ```bash
    fdisk /dev/vdb
    ```
::: info
MBR 分区表请使用 fdisk 分区， GPT 分区
表请使用 gdisk 分区
:::
- `blkid` 查看设备UUID等参数
- `parted` 列出磁盘的分区表类型与分区信息
    **example input**
    ```bash
    parted /dev/vdb print
    ```

## 磁盘分区步骤
- 通过 lsblk 或 blkid 先找到磁盘
- 再用 parted /dev/xxx print 来找出内部的分区表类
型
- 才用 gdisk 或 fdisk 来操作系统

## 创建文件系统

`mkfs` make filesystem
CentOS 7 使用 xfs 作为默认文件系统
```bash
mkfs.xfs /dev/vdb1
```

### 文件系统挂载
将文件系统挂载到目录上，目录是文件系统的入口
::: warning 
- 单一文件系统不应该被重复挂载在不同的挂载点（目录）中；
- 单一目录不应该重复挂载多个文件系统；
- 要作为挂载点的目录，理论上应该都是空目录,否则原文件系统下的东西会隐藏
:::
- `mount` 挂载文件系统
    **example input**
    ```bash
    mount /dev/vdb1 /mnt
    ```
